# 🚀 GST Exam Countdown - Ultra Premium

একটি আধুনিক, এনিমেটেড এবং ফিচার-সমৃদ্ধ **GST ভর্তি পরীক্ষার কাউন্টডাউন** ওয়েব অ্যাপ্লিকেশন। এটি HTML, CSS এবং JavaScript দিয়ে তৈরি করা হয়েছে যা শিক্ষার্থীদের পরীক্ষার সময় সম্পর্কে সচেতন রাখতে সাহায্য করবে।

---

## ✨ ফিচারসমূহ (Features)

* **বিসমিল্লাহির রাহমানির রাহিম:** শুরুতে ধর্মীয় বরকত দিয়ে ডিজাইন শুরু।
* **রিয়েল-টাইম এনালগ ক্লক:** HTML5 Canvas ব্যবহার করে তৈরি একটি প্রিমিয়াম এনালগ ঘড়ি।
* **ডাইনামিক কাউন্টডাউন:** পরীক্ষার কত দিন, ঘণ্টা, মিনিট এবং সেকেন্ড বাকি আছে তা লাইভ আপডেট হয়।
* **বাংলা ডে ডিসপ্লে:** বর্তমান দিনটি বাংলায় দেখানোর ব্যবস্থা (যেমন: আজ শুক্রবার)।
* **গুগল ড্রাইভ ডাউনলোডার:** ড্রাইভের লিঙ্ক দিলে সরাসরি ফাইল ডাউনলোডের সুবিধা।
* **আল্ট্রা প্রিমিয়াম ডিজাইন:** গ্লাস-মরফিজম (Glassmorphism) এবং এনিমেটেড ব্যাকগ্রাউন্ড ব্লবস (Blobs)।

---

## 🛠️ টেকনোলজি স্ট্যাক (Tech Stack)

* **Frontend:** HTML5, CSS3 (Advanced Animations)
* **Logic:** JavaScript (ES6+)
* **Graphics:** HTML5 Canvas API (Clock logic)
* **Design:** Modern Glassmorphism UI

## 🚀 যেভাবে ব্যবহার করবেন (How to Use)

১. আপনার পিসিতে রিপোজিটরটি ক্লোন করুন অথবা জিপ ফাইল ডাউনলোড করুন।
২. `index.html` ফাইলটি যেকোনো ব্রাউজারে (Chrome, Edge, Firefox) ওপেন করুন।
